# 프로젝트 실행방법
1. 사이트 우측상단에 있는 signup에서 회원가입을 한다.
2. login에서 로그인을 한다.
3. 로그인 후, todo페이지에서 내가 할일을 작성한다.(todo페이지로 자동으로 안가질경우, 좌측 상단에 있는 Home클릭)
4. 내가 할일을 등록한다.
5. 할일을 완료했으면 완료체크를 할 수있다.
6. 할일을 수정하거나 삭제를 할 시, 할일 우측에 있는 버튼을 활용한다.
